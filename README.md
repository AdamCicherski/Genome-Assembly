## General info
This project is simple genome de Novo assembler based on de Brujin Graphs aproach
	
## Technologies
Project is created with:
* Numpy >=1.21.1
* Numba >= 0.54.0
	
## Quick start
To run genome assembler type:

```
$ python assembly.py reads.fasta output.fasta
```
